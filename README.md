# 🦟 臺北市登革熱病媒蚊密度監測儀表板

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live%20Demo-brightgreen)](https://tylin23.github.io/taipei_dengue_fever/)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

這是一個互動式網頁儀表板，用於展示和分析臺北市登革熱病媒蚊密度調查結果。提供直觀的資料視覺化和分析功能。

## ✨ 功能特色

### 🎯 核心功能
- 📊 **動態儀表板**: 動態載入和更新資料，支援多月份、多行政區動態範圍
- 🔍 **智慧篩選**: 月份、行政區雙重篩選，即時更新所有視圖
- 🗺️ **互動式地圖**: SVG 向量地圖，支援點擊查看和選擇性顯示
- 📈 **多元圖表**: 趨勢線圖、長條圖、圓餅圖等多種視覺化
- 📋 **動態表格**: 支援項目排序功能

### 🎨 使用者體驗
- 📱 **完整 RWD**: 支援桌面、平板、手機等各種裝置
- 📊 **載入回饋**: 即時進度條和載入狀態顯示
- 🔄 **即時比較**: 自動計算月份間變化百分比
- 💡 **智慧提示**: 點擊地圖顯示篩選條件下的詳細統計

### 📊 資料分析
- 🏠 **調查統計**: 總調查戶數、陽性戶數統計
- 📉 **風險評估**: 布氏指數和容器指數分析
- 📅 **趨勢追蹤**: 跨月份數據比較和變化趨勢
- 🎯 **風險分級**: 自動標示安全、低風險、中風險、高風險等級

## 📊 資料格式

CSV 檔案包含以下欄位：
- **日期**: 調查日期
- **區別**: 行政區名稱
- **里別**: 里名稱
- **調查戶數**: 調查的住戶數量
- **陽性戶數**: 發現病媒蚊的住戶數量
- **布氏指數**: 病媒蚊密度指標
- **容器指數**: 容器陽性率指標
- **調查種類**: 調查類型 (住宅/菜園/學校等)

## 🎨 自定義和擴展

### 添加新資料
1. 將 CSV 檔案放入 `data/` 目錄
2. 檔案命名格式：`臺北市{年份}年{月份}月登革熱病媒蚊密度調查結果表.csv`
3. 系統自動檢測並載入

### 修改樣式
- 編輯 `styles.css` 調整視覺效果
- 支援 CSS 變數，易於主題切換
- 完整的 RWD 斷點設定

## 🎮 使用方式

1. **選擇時間範圍**: 選擇要查看的月份（可選擇全部月份或特定月份）
2. **選擇區域**: 選擇要查看的行政區（可選擇全部行政區或特定行政區）
3. **互動式地圖**: 點擊地圖上的行政區可快速篩選該區域資料
4. **智慧 Tooltip**: 點擊行政區顯示當前篩選條件下的詳細統計
5. **資料分析**: 查看各種圖表了解病媒蚊密度趨勢和分布
6. **詳細資料**: 使用固定表頭的詳細資料表進行深度分析

### 💡 使用技巧
- **快速篩選**: 先選擇月份，再點擊地圖查看該月份特定行政區的數據
- **比較分析**: 切換不同月份，觀察同一行政區的數據變化
- **表格操作**: 點擊表頭進行排序，使用固定表頭方便瀏覽大量資料
